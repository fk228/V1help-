# V1 Help

A lightweight Windows utility designed to simplify common system tasks through an easy-to-use interface.

## Features

- System Information
- File Hash Checker
- Windows Utilities
- Useful Maintenance Tools
- Beginner-Friendly Interface
- Fast and Lightweight

> Note: The Cleanup feature is currently unavailable and will be fixed in a future update.

## Installation

1. Download `install V1help.exe`
2. Run the installer.
3. Follow the installation steps.
4. Launch the program from the Start Menu or Desktop.

## Requirements

- Windows 10 / Windows 11
- Administrator privileges may be required for some features.
